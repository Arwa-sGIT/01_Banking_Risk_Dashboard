# Data Dictionary

| Column Name | Description |
|-------------|-------------|
| `Loan_ID` | Unique loan identifier |
| `Loan_Amount_EUR` | Original loan amount |
| `Outstanding_Balance_EUR` | Current amount owed |
| `Sector` | Client's industry |
| `Credit_Rating` | Internal credit assessment (AAA to CCC) |
| `Days_Past_Due` | Days late on payments (0 = Performing) |
| `KYC_Status` | Compliance status (Valid, Expiring Soon, Expired) |
